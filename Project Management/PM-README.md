# Project Management Hub

This folder contains planning and tracking documents for Dimensional Dex.

## Core Documents
- Project Proposal.md
- Scope.md

## Sprint Documents
- Sprint Planning/SP1.md to Sprint Planning/SP7.md
- Sprint Retro/SR1.md to Sprint Retro/SR7.md

## Management Templates
- Milestones.md
- Task Board.md
- Risks-Issues.md
- Decisions.md
- Test Checklist.md
- Demo Script.md

## How to Use
1. Start each day by updating Task Board.md and the current sprint planning file.
2. Record any blockers in Risks-Issues.md as soon as they appear.
3. Log technical choices in Decisions.md when you make them.
4. Run through Test Checklist.md before each major checkpoint.
5. Use Demo Script.md to rehearse your final presentation.
